Here are some of my projects

  • Supreme Therapy

    React, Javascript, HTML, CSS, MongoDB

    I developed a comprehensive website for a physical therapy clinic that streamlines appointment booking and profile management for patients, therapists, and administrators. The platform features a user login, allowing patients to fully customize their profiles and book appointments with therapists in their area. Additionally, I implemented a doctor login, enabling therapists to access their scheduled appointments and personalize their profiles. An admin login was also integrated, giving the clinic owner full oversight of all booked appointments and the ability to add new doctors to the system as needed. This platform enhances efficiency and provides a seamless experience for both patients and staff.

    Treact Project

    HTML, CSS, JavaScript

    For my project, I embarked on replicating the Treact website, a platform renowned for its sleek design and user-friendly interface. The challenge involved meticulously analyzing the site's layout, functionality, and interactive elements to recreate them with precision. I focused on mirroring the clean aesthetic and seamless navigation, employing modern web technologies to ensure a responsive and engaging user experience. This project not only honed my skills in front-end development but also deepened my understanding of design principles and web optimization techniques.

    Library React App

    HTML, CSS, JavaScript, React

    I developed a dynamic and user-friendly online bookstore that allows users to seamlessly browse and purchase books. The website features an add-to-cart system, enabling users to select multiple books before proceeding to checkout. Additionally, I implemented a skeleton loading state, improving the user experience by displaying placeholders while content loads, ensuring a smoother and more visually appealing interface. The platform is designed with an intuitive layout for an efficient and enjoyable shopping experience.

    Netflix Clone

    HTML, CSS, JavaScript, React, Firebase

    In this project, I developed a clone of the Netflix website using React, Firebase, and various APIs. The application mimics the user interface and functionality of Netflix, allowing users to browse a catalog of movies and TV shows, view details, and manage their watchlist. Firebase provides real-time data synchronization and authentication, while React handles the dynamic rendering of content and state management. The project showcases the use of modern web technologies to replicate a popular streaming platform's user experience.
